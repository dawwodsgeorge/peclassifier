# PEClassifier Support

PEClassifier is a clinical reference tool implementing the 2026 AHA/ACC/ACCP/ACEP/CHEST acute pulmonary embolism classification system, including the full HESTIA criteria checklist.

## Frequently Asked Questions

**What is PEClassifier for?**
PEClassifier helps clinicians and trainees quickly apply the current PE severity classification (Categories A–E with sub-stages) based on the latest guideline criteria.

**Is my data stored anywhere?**
No. Any clinical history summaries you save stay on your device only and are never transmitted anywhere. See the [Privacy Policy](./privacy-policy.html) for details.

**Is this a diagnostic tool?**
No. PEClassifier is intended as an educational and reference aid for licensed healthcare professionals. It does not replace clinical judgment or established diagnostic protocols.

**I found a bug or have a suggestion.**
Please reach out using the contact info below — feedback on classification accuracy against the current guideline is especially welcome.
